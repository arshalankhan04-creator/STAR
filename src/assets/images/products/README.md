# Product Images

Drop all product images in this folder.

## Recommended naming convention

| Product | Suggested filename |
|---|---|
| Skin Care Soap | `soap-skin-care.jpg` |
| Aloevera Cool Soap | `soap-aloevera-cool.jpg` |
| Charcoal & Multani Mitti Soap | `soap-charcoal-multani.jpg` |
| Baby Soap | `soap-baby.jpg` |
| Besan & Pithi Soap | `soap-besan-pithi.jpg` |
| Honey Soap | `soap-honey.jpg` |
| Nim Aloevera Soap | `soap-nim-aloevera.jpg` |
| Milky Soap | `soap-milky.jpg` |
| Star Facial Soap | `soap-star-facial.jpg` |
| Herbal Oil | `herbal-oil.jpg` |
| Herbal Shampoo | `herbal-shampoo.jpg` |
| Herbal Face Gel | `herbal-face-gel.jpg` |

## Supported formats
JPG, PNG, or WebP. JPG recommended for photos.

## After adding images
Update the `image` field in `src/data/products.js` like this:

```js
image: '/src/assets/images/products/soap-milky.jpg',
```
